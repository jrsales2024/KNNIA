O documento apresenta um roteiro para uma aula prática sobre a aplicação de aprendizado supervisionado em visão computacional, utilizando o classificador KNN.
Objetivos da Aula Prática

Executar um código em Python para classificar imagens de dígitos utilizando o classificador KNN.
Compreender conceitos e técnicas de machine learning na classificação de imagens.

Procedimentos Práticos

Iniciar o Google Colab e criar um novo notebook.
Importar bibliotecas necessárias e acessar o dataset de dígitos do Scikit-learn.
Visualizar a estrutura do dataset e uma das imagens.
Criar um DataFrame para melhor visualização dos dados.
Dividir o conjunto de dados em treino (20%) e teste (80%).
Aplicar o classificador KNN com K=5 e exibir a matriz de confusão e métricas de avaliação.
Alterar o valor de K e observar o impacto nos resultados.

Resultados Esperados

Entregar um arquivo 'knn_ai.py' com as funções implementadas e um PDF com prints das saídas dos exercícios.
Ao final, os alunos devem ser capazes de compreender o processo de treinamento e classificação de dados, além de utilizar ferramentas básicas de desenvolvimento em Python.
